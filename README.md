# goPassGen

This is a simple Go script that can generate a list of easily-guessable passwords for further Password Spray Attack.

## Installation
```bash
git clone https://github.com/bigb0sss/goPassGen.git
```
## Usage & Example
#### Option#1
```bash
go build goPassGen.go
./goPassGen
```
#### Option#2
```bash
go run goPassGen.go
```
